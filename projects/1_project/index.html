M5HESTIA Project

MIMO and M-MIMO for 5G application

M5HESTIA project aims at designing advanced M-MIMO antennas, characterizing / modelling the outdoor mmW channel and proposing innovative algorithms in order to demonstrate, a full M-MIMO hardware (HW) platform operating in the 60-GHz band. For more informations at M5HESTIA Cominlab funding by Région Bretagne/B<>com/Orange/IMT-Atlantique.

Project consists of 5 work parkages; Some of these packages has been used HFSS/CST/ADS tool and Python program.

  • WP2: antenna design and RF architecture
  • WP4: baseband signal processing
